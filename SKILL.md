# qi-cli — Qi 语言命令行框架

用 Qi（奇语）写的 CLI 框架，风格参考 spf13/cobra：命令树、子命令、局部/持久参数、别名、执行钩子、自动帮助。底层参数解析复用 `标准库.命令行`。

> **先读 `qi-lang` 技能**。本技能只讲 qi-cli 的 API。

## 何时使用

- 用户用 Qi 写命令行工具
- 用户问 qi-cli 的命令、子命令、参数、标志、钩子
- 用户提到 `导入 CLI`、`创建应用`、`创建命令`

## 三个核心类型

- `应用` — 顶层 CLI 应用
- `命令` — 命令 / 子命令
- `参数` — 标志或位置参数
- `上下文` — 在 handler 里读参数/标志

**handler 签名固定**：`函数(上下文): 整数`（返回退出码，0=成功）。
**链式 API**：每个 setter 返回新对象，必须回写：`应用值 = 应用版本(应用值, "1.0")`。

## 最小示例

```qi
包 主程序;

导入 标准库.输入输出 作为 IO;
导入 CLI::{
    创建应用, 创建命令, 创建参数, 执行,
    应用简介, 应用执行, 应用添加命令,
    命令简介, 命令执行, 命令添加参数,
    参数帮助, 值
};

函数 执行问候(上下文值: 上下文) : 整数 {
    变量 名字: 字符串 = 值(上下文值, "名字");
    如果 (字符串::字节长度(名字) == 0) { 名字 = "世界"; }
    IO.打印行("你好，" + 名字 + "！");
    返回 0;
}

函数 根执行(上下文值: 上下文) : 整数 {
    IO.打印行("用 --help 查看子命令");
    返回 0;
}

函数 入口() {
    变量 应用值: 应用 = 创建应用("工具箱");
    应用值 = 应用简介(应用值, "一个 Qi CLI 示例");
    应用值 = 应用执行(应用值, 根执行);

    变量 问候命令: 命令 = 创建命令("问候");
    问候命令 = 命令简介(问候命令, "向某人打招呼");
    问候命令 = 命令执行(问候命令, 执行问候);

    变量 名字参数: 参数 = 创建参数("名字");
    名字参数 = 参数帮助(名字参数, "要问候的名字");
    问候命令 = 命令添加参数(问候命令, 名字参数);

    应用值 = 应用添加命令(应用值, 问候命令);
    执行(应用值);   // 解析 argv 并分发
}
```

## API 速查

### 应用
`创建应用(名)` · `应用版本` · `应用作者` · `应用简介` · `应用说明` · `应用用法` · `应用示例` · `应用执行(应用,处理器)` · `应用添加参数` · `应用添加命令`

### 命令
`创建命令(名)` · `命令简介` · `命令说明` · `命令用法` · `命令示例` · `命令添加别名` · `命令执行` · `命令添加参数` · `命令添加子命令`
钩子：`命令前置执行` · `命令后置执行` · `命令持久前置执行` · `命令持久后置执行`（持久钩子会传递到所有子命令，且拿到最深匹配命令的上下文）

### 参数
`创建参数(名)` · `参数长名(p,"详细")` · `参数短名(p,"v")` · `参数帮助` · `参数默认值` · `参数必需` · `参数标志`（布尔开关）· `参数持久`（继承到子命令，走 clap global）

### 上下文（在 handler 里读）
```qi
变量 文本值: 字符串 = 值(上下文值, "名字");
变量 数: 整数 = 整数值(上下文值, "端口", 3076);     // 带默认值
变量 开关: 整数 = 标志(上下文值, "详细");           // 0/1
变量 有否: 整数 = 存在值(上下文值, "名字");
变量 路径: 字符串 = 命令路径(上下文值);             // "应用 命令 子命令"
```

## 子命令树

```qi
变量 用户命令: 命令 = 创建命令("用户");
变量 添加命令: 命令 = 创建命令("添加");
添加命令 = 命令执行(添加命令, 执行添加);
用户命令 = 命令添加子命令(用户命令, 添加命令);   // 工具箱 用户 添加
应用值 = 应用添加命令(应用值, 用户命令);
```

## 标志参数注意

- `参数标志` 的参数若没设 long/short，框架自动用参数名作长名（clap 要求 flag 必须有名）
- 持久参数 (`参数持久`) 用 clap 的 global 机制传到子命令，**不要**手动复制到每个子命令

## 运行

```bash
qi run 工具箱.qi 问候 小李
qi run 工具箱.qi --help
qi run 工具箱.qi 用户 添加 小王
```

## 端口约定

示例里涉及端口用随机高位端口（`3076` / `6759` / `43510`），不要用 8080 / 3000。
